# Egglog encoding

This is a repo for the egglog encoding project, which maintain a fork of egglog and egglog experimental as sub-directories, so that we can iterate more quickly on changes to them without forking main.

In the end we want to port whatever our "winning" strategy is back to main egglog and experimental.

The first goal is get proofs working with a reasonable slowdown (<2x) compare to main
